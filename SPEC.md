# 个人作品集网站规范

## 1. 项目概述
- **项目名称**：Portfolio
- **类型**：单页作品集展示网站
- **核心功能**：展示海报、公众号文章、小红书文章三类作品
- **目标用户**：展示个人创作成果

## 2. UI/UX 规范

### 2.1 布局结构
- **头部**：固定顶栏，包含网站标题和分类Tab
- **主体**：网格画廊布局展示作品
- **模态框**：点击作品查看大图/详情

### 2.2 响应式断点
- 移动端：< 768px（2列网格）
- 平板：768px - 1024px（3列网格）
- 桌面：> 1024px（4列网格）

### 2.3 视觉设计（米色系简洁风）
- **主色**：#f5f0e8（米白）
- **强调色**：#c9a87c（暖金）
- **背景色**：#faf8f5（米色）
- **卡片背景**：#ffffff（纯白）
- **文字色**：#333333（深灰）
- **次要文字**：#888888（灰色）
- **边框色**：#e8e4dc（浅米灰）
- **字体**：
  - 标题：Noto Serif SC, serif
  - 正文：Noto Sans SC, sans-serif

### 2.4 组件
- **顶部导航**：固定，极简线条分割
- **分类Tab**：三列 tabs（海报/公众号/小红书），选中态底部细线
- **作品卡片**：
  - 图片封面（aspect-ratio 4:3）
  - 悬浮时显示标题
  - 悬停动画：轻微上浮 + 边框加深
- **模态框**：
  - 居中显示
  - 半透明米色遮罩
  - 关闭按钮
  - 图片/文章链接

### 2.5 动画效果
- 页面加载：卡片依次淡入（stagger 50ms）
- Tab切换：内容区淡入淡出
- 卡片悬停：transform: translateY(-4px)，过渡 0.25s ease

## 3. 功能规范

### 3.1 核心功能
- Tab切换三个分类
- 作品网格展示
- 点击查看大图/详情
- 显示作品标题、描述、日期

### 3.2 数据结构
```javascript
{
  id: string,
  title: string,
  description: string,
  image: string,  // 图片URL
  date: string,   // 发布日期
  link: string,   // 原文链接（可选）
  tags: string[]  // 标签
}
```

### 3.3 作品管理
- 所有作品数据存放在 `src/data/works.js`
- 每天添加新作品，只需编辑这一个文件
